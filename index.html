OurMods is now closed..

Scope

The OurMods project seeked to preserve the rich history of Minecraft Modding. Be it mods, plugins, server administration tools, mod loaders, world modification tools, game maps and beyond. We didn't seek to serve strictly as a "mod host", but as an archive of the mod content that makes Minecraft so unique and versatile.

Why?

Mod Authors relied on a connection to sources like Dropbox and Mediafire. When these companies eventually decide to change how they operate and links start going down. The mod will become unplayable and unable to be enjoyed by future generations. By archiving these mod files, along with additional community generated content. The lost content will then be available to play long aftertheir developers have abandoned them.

What We Archived

  • All libraries and resources (often downloaded at runtime) needed to launch each vesion of a given game.
  • Every single version of Forge, Modloader, ModloaderMP, MinecraftPortCentral(Beta 1.8.1 â€” 1.2.5), Cauldron Craftbukkit, Spigot. Even jenkins builds, development builds, and experimental builds.
  • Guides for each version of mods and plugins, including installation and use.
  • All file downloads as torrents, or as direct downloads from our file server (with checksum verification).
  • Git mirrors: The latest available git respositories for all opened source mods, plugins, and tools. Hosted on our servers in a read-only state, and included in the OurMods Insurance torrents.
  • Monthly full-site dumps in torrent format, akin to WikiLeaks' Insurance File. If the site ever shuts down, a recent version of the entirety of the site's content (files, guides, even some site code) should be avaiable on the web forever, or at least as long as someone is seeding.
  • A Full featured API - Allowing the community to create Launchers, IRC Bots, or anything else you might come up with.

What Now?

  • Old/Discontinued mods will be obtained by contacting the original developer, searching through old modpack archives, and by community-sourcing via IRC/Reddit.
  • Recent mods will be added manually as well, and by scraping existing mod hosting services.
  • Contributors will create curated pages and community forums for each mod.

Get Involved

Our Mods has shut down, but our legacy stil exists. If you'd like to assist in the archival process, please join the MCArchive Discord Server

MCArchive really appreciates help finding legitimate files for mods and plugins that are no longer available due to dead links, thread closings, Fandom's Minecraft Forge Purge, Dropbox Public Folders, etc. If you find any such files, MCArchive will love to hear about it.
